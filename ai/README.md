# AI Toolkit

Enhanced AI capabilities with Gemini and Perplexity integrations.

## What's Included

### MCP Servers (2)
- **gemini-bridge** - Google Gemini AI model integration
- **perplexity** - AI-powered search and information retrieval

## Installation

```bash
/plugin install ai-toolkit@wookstar
```

## Required Environment Variables

Create a `.env` file in your project root:

```bash
# Perplexity (required)
PERPLEXITY_API_KEY=your_key_here
```

## API Keys

- **Perplexity**: Get API key at https://www.perplexity.ai/settings/api
- **Gemini**: Configured via uvx, no separate key needed

## Usage Examples

```bash
# Use Perplexity for research
"Search for the latest developments in quantum computing"
"Find recent news about AI regulations"
"What are the current trends in renewable energy?"

# Gemini integration
"Use Gemini to analyze this image"
"Compare Gemini's response with Claude's for this task"
```

## When to Use

This toolkit is ideal for:
- AI-powered research and information gathering
- Multi-modal tasks (text and images)
- Real-time information retrieval
- Comparing AI model outputs
- Searching for latest news and developments
