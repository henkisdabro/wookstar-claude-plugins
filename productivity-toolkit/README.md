# Productivity Toolkit

Essential productivity and planning tools for all types of work in Claude Code.

## What's Included

### Agent (1)
- **documentation-manager** - Technical documentation specialist for README files, API docs, and architecture decisions

### Commands (11)
- **/containerize** - Create production-ready Docker configurations
- **/prompt_writer** - Advanced prompt engineering assistant
- **/planning** - Structured planning workflows for feature development
- **/ultra-think** - Deep analytical thinking for complex problems
- **/generate-prp** - Generate Progressive Refinement Plans
- **/execute-prp** - Execute previously generated PRPs
- **/infinite** - Extended context planning for large-scale projects
- **/prep-parallel** - Prepare tasks for parallel execution
- **/execute-parallel** - Execute prepared tasks in parallel
- **/reflection** - Code analysis and reflection
- **/primer** - Quick project initialization

### Skills (2)
- **prp-generator** - Generate comprehensive Product Requirement Plans
- **timezone-tools** - Timezone conversion and time utilities

## Installation

```bash
/plugin install productivity-toolkit@wookstar
```

## Usage Examples

```bash
# Plan a complex feature
/ultra-think "Design a real-time collaboration system"

# Generate a comprehensive plan
/generate-prp "Build user authentication module"

# Execute the plan
/execute-prp

# Get timezone help
"What time is it in Tokyo?"

# Container-ize your application
/containerize
```

## When to Use

This toolkit is ideal for:
- Project planning and architecture design
- Complex problem-solving that requires deep thinking
- Creating comprehensive implementation plans
- Containerizing applications
- Managing timezones in distributed teams
- Generating product requirement documents
